# Bias-Analysis-of-Cardiac-Disease-Prediction-Models-Across-Gender-Age-and-Ethnicity
 This is a Data science project that uses techniques to examine bias in cardiac disease prediction models across gender, age, and ethnicity.
The first component focuses on predicting cardiac disease using machine learning models trained on both behavioural and clinical variables. 
The second component evaluates bias and fairness by analysing data from the Institute of Health Metrics and Evaluation (IHME) and the Behavioral Risk Factor Surveillance System (BRFSS).
By integrating these approaches, the project aims to deliver both accurate predictive insights and a broader regional understanding of cardiac disease patterns.
